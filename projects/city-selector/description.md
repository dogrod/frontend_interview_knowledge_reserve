# 完成一个城市选择组件（网易）

## 要求

1. 可定位置当前所在城市，可支持传城市。
2. 下次打开优先取上次定位城市，如本次定位和上次不一样，则取本次城市，同时展示最近选择的城市，最近选择的城市可配。
3. 城市列表按字母分组，如B组：北京、包头，同时左侧带a-z导航浮条，点击对应字母定位至对应的组位置，如点击C则定位至C组，同时页面弹出提示为C。
4. 支持城市搜索，页头带搜索框，可支持联想功能，注意性能。
5. 支持对应城市，会将对应城市数据带回给使用页面。
6. 支持单个页面上同时存在多个城市组件。
7. 页面使用flex布局。（CSS）
